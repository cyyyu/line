# Line

Dead simple line chart in webgl.

<img src="./screenshot.png" width="280" />

### Usage

`npm install --save simple-line-chart`

```
import line from "simple-line-chart";

line({
  canvas: document.querySelector("canvas"),
  data: [100, 200, 300, 200, 400, 100]
});
```
